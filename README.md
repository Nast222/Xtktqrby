# Random Password Generator

**Автор:** Челейкин Алексей

## Описание
Простое GUI-приложение для генерации случайных паролей с настройкой длины и состава символов. История паролей сохраняется в файл `history.json`.

## Требования
- Python 3.x
- Библиотеки: tkinter, random, json, pyperclip (для копирования)

## Установка
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourname/PasswordGenerator.git
   ```
2. Установите зависимости (если нужно):
   ```
   pip install -r requirements.txt
   ```
3. Запустите:
   ```
   python main.py
   ```

## Использование
1. Выберите длину пароля ползунком.
2. Отметьте нужные типы символов.
3. Нажмите «Сгенерировать».
4. Пароль появится в таблице истории.
5. Двойной клик по паролю скопирует его в буфер обмена.

## Примеры тестов
- Генерация пароля длиной 8, только цифры.
- Генерация пароля длиной 16, все типы символов.
- Проверка сохранения и загрузки истории после перезапуска.
